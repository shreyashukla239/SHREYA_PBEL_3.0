# Smart Attendance Portal

This is a contactless attendance monitoring system. It uses modern facial recognition technology to identify students and record check-ins automatically.

## Key Features

- **Facial Check-In**
- **Oval Viewfinder Guide**
- **Student Profiles**
- **Class Check-In**
- **Streak Tracker**
- **Absentee Panel**
- **System Telemetry**
- **Spreadsheet Exports**
